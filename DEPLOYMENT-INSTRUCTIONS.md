## 🛠️ **Local Setup Instructions**

### **Step 1: Clone Your Repository**
```bash
git clone https://github.com/Abhishek-7j/Secure-Browser.git
cd Secure-Browser
```

### **Step 2: Install Dependencies**
```bash
npm install
```

### **Step 3: Setup Database**
```bash
# PostgreSQL (Recommended)
npm run db:init

# Or use SQLite (automatic fallback)
```

### **Step 4: Start Browser**
```bash
npm start
```

### **Step 5: Access Browser**
- **Local**: Opens automatically in Electron
- **Web**: Available at http://localhost:3000

## 🔧 **Configuration**

### **Environment Setup**
```bash
# Copy environment file
cp .env.example .env

# Edit settings
DB_HOST=localhost
DB_PORT=5432
DB_NAME=faprot_browser
DB_USER=postgres
DB_PASSWORD=your_password
```

### **Security Settings**
```env
SECURITY_LEVEL=high
AUTO_BLOCK_SUSPICIOUS=true
HTTPS_ONLY=true
BLOCK_TRACKERS=true
AI_THREAT_DETECTION=true
```
